# Backend PocketBase — Deploy API

Esta pasta contém a estrutura de deploy do backend PocketBase do projeto **Escola Bíblica 360**.

O backend é responsável por sustentar:

- painel administrativo;
- autenticação;
- posts;
- categorias;
- downloads;
- newsletter;
- mensagens de contato;
- biblioteca de mídia;
- logs;
- uploads;
- regras de acesso;
- migrations do banco.
